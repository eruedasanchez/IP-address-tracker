### Run Locally

To run **Time Tracking Dashboard** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/eruedasanchez/IP-address-tracker
```

Windows:

```bash
git clone https://github.com/eruedasanchez/IP-address-tracker
```
